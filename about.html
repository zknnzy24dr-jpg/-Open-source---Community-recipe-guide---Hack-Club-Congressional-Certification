Cooking shouldn't require a perfect pantry.

Community Recipe Guide started from a simple observation: most recipe sites assume you're shopping for a specific dish. But for a lot of families, the real question is the opposite one — "what can I make with what I already have?" That question matters most for households navigating tight grocery budgets, food deserts, or unpredictable income, where an extra trip to the store isn't always possible.

How the recipe generator works

You list the ingredients you have on hand. The generator checks them against a database of recipes built around shelf-stable, low-cost staples — rice, beans, lentils, pasta, eggs, canned tomatoes — and ranks results by how well they match what you already own, then by estimated cost per serving. Cheaper, more complete matches surface first.

The matching logic is fully rule-based and runs entirely on the server with no external API calls, so the tool works offline, loads instantly, and never depends on a third-party AI service being available.

Why health equity

Nutrition guidance often assumes resources that not every household has. By starting from "what do you already have" rather than "here's what you need to buy," the guide tries to meet people where they are and reduce the friction between wanting to cook a healthy meal and actually being able to.

About this submission

This project was built as a Congressional App Challenge submission using Python and Flask. The full source code is open and available for review — see the project README for setup instructions and an overview of how each part of the application works.
